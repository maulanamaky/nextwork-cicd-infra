## This is NextWork Web Project: 6 Day DevOps Challenge!

You can also join the challenge: https://learn.nextwork.org/projects/aws-devops-cicd

This is my version to convert from Console to IaC with Terraform.

How To Run?

1. You must make new file `terraform.tfvars` and fill it with your SSH public key.
2. This is not have backend, its supposed to be the state in local.
3. Make 2 Repository Github, 1 for Developer Setup IaC and 1 for EC2 Production.
